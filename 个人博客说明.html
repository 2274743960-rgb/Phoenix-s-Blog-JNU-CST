项目概述 / Project Overview

中文
English

Phoenix的计算机科学学习旅程是一个交互式学习管理平台，专为计算机科学专业学生设计，帮助他们系统化地规划、跟踪和管理学习进度。

该项目由夏国庆（Phoenix）开发，他是暨南大学国际学院计算机科学与技术专业2025届的学生。平台整合了学习模块、进度跟踪、任务管理、成就系统等功能，为学生提供了一个全面的学习辅助工具。

项目完全使用HTML、CSS和JavaScript实现，无需后端支持，数据通过浏览器本地存储保存。

Phoenix's Computer Science Learning Journey is an interactive learning management platform designed specifically for computer science students to systematically plan, track, and manage their learning progress.

This project was developed by Guoqing Xia (Phoenix), a 2025 Computer Science and Technology student at Jinan University International College. The platform integrates learning modules, progress tracking, task management, achievement systems, and more, providing students with a comprehensive learning assistance tool.

The project is entirely implemented using HTML, CSS, and JavaScript, requiring no backend support, with data saved through browser local storage.

设计初衷 / Design Philosophy

中文
English

作为一名计算机科学专业的学生，我深刻理解学习过程中面临的挑战：

  • 知识体系庞大，难以系统化学习
  • 学习任务繁多，缺乏有效跟踪
  • 进度难以可视化，缺乏成就感
  • 理论与实践结合不够紧密

基于这些痛点，我设计了这款学习旅程平台，旨在：

  • 提供结构化的学习路径导航
  • 实现学习进度的可视化跟踪
  • 通过成就系统增强学习动力
  • 建立知识体系间的联系

设计理念： 将学习过程游戏化，通过即时反馈和奖励机制激发学习兴趣，帮助学生建立持续学习的习惯。

As a computer science student, I deeply understand the challenges faced during the learning process:

  • Vast knowledge system makes systematic learning difficult
  • Numerous learning tasks lack effective tracking
  • Progress is hard to visualize, lacking a sense of achievement
  • Insufficient integration of theory and practice

Based on these pain points, I designed this learning journey platform aiming to:

  • Provide structured learning path navigation
  • Achieve visual tracking of learning progress
  • Enhance learning motivation through achievement systems
  • Establish connections between knowledge systems

Design Philosophy: Gamify the learning process, stimulate learning interest through instant feedback and reward mechanisms, and help students develop consistent learning habits.

使用说明 / User Guide

中文
English

快速开始

  1. 下载项目文件或克隆代码库
  2. 在浏览器中打开 index.html 文件
  3. 开始探索和使用各项功能

功能使用指南

1. 学习模块导航

使用左侧导航栏浏览不同的学习模块，点击模块名称查看详细内容。

2. 进度跟踪

完成学习任务后，勾选对应的复选框或点击"完成"按钮更新进度。

3. 每日打卡

点击"今日打卡"按钮记录每日学习，保持连续学习天数。

4. 个人资料设置

点击"上传照片"按钮更换个人头像。

5. 学习资源访问

点击资源卡片中的链接访问外部学习资源。

提示： 所有数据都保存在浏览器本地存储中，清除浏览器数据将导致学习记录丢失。

Quick Start

  1. Download the project files or clone the code repository
  2. Open the index.html file in a browser
  3. Start exploring and using all features

Feature Usage Guide

1. Learning Module Navigation

Use the left navigation bar to browse different learning modules, click on module names to view detailed content.

2. Progress Tracking

After completing learning tasks, check the corresponding checkboxes or click the "Complete" button to update progress.

3. Daily Check-in

Click the "Check-in Today" button to record daily learning and maintain consecutive learning days.

4. Profile Settings

Click the "Upload Photo" button to change your profile picture.

5. Learning Resource Access

Click links in resource cards to access external learning resources.

Tip: All data is saved in browser local storage, clearing browser data will result in loss of learning records.

1. 数据持久化

使用浏览器的localStorage API保存学习数据：

// 保存数据到本地存储 function saveData() { localStorage.setItem('learningData', JSON.stringify(learningData)); } // 从本地存储加载数据 function loadData() { const savedData = localStorage.getItem('learningData'); if (savedData) { learningData = JSON.parse(savedData); } updateDisplay(); }

2. 响应式布局

使用CSS Grid和Flexbox实现响应式布局：

.dashboard { display: grid; grid-template-columns: 300px 1fr; gap: 2rem; } @media (max-width: 992px) { .dashboard { grid-template-columns: 1fr; } }

function initParticles() { const canvas = document.getElementById('particleCanvas'); const ctx = canvas.getContext('2d'); // 设置画布尺寸 canvas.width = window.innerWidth; canvas.height = window.innerHeight; // 粒子数组和动画循环 // ... }

4. 进度计算和更新

动态计算和更新学习进度：

function updateDisplay() { // 计算总体进度 let totalModules = 0; let completedModules = 0; for (const module in learningData.modules) { totalModules += learningData.modules[module].total; completedModules += learningData.modules[module].completed; } const overallProgress = totalModules > 0 ? 自定义指南 / Customization Guide

中文
English

个性化学习路径

您可以轻松修改学习路径以适应自己的学习需求：

1. 修改导航菜单

在HTML中找到导航菜单部分，修改模块名称和结构：

<ul class="nav-menu"> <li> <a href="#" onclick="showModule('foundation')">1. 编程基础</a> <ul> <li><a href="#" onclick="showSubModule('python-basics')">1.1 Python基础</a></li> <li><a href="#" onclick="showSubModule('java-fundamentals')">1.2 Java基础</a></li> <!-- 添加更多子模块 --> </ul> </li> <!-- 添加更多主模块 --> </ul>

2. 更新学习模块

在learningData对象中添加或修改学习模块：

let learningData = { modules: { foundation: { completed: 0, total: 4, status: 'not-started' }, dataStructures: { completed: 0, total: 4, status: 'not-started' }, // 添加新模块 newModule: { completed: 0, total: 5, status: 'not-started' } }, // ... };

3. 自定义颜色主题

修改CSS变量来更改整体配色方案：

:root { --primary-color: #2c3e50; /* 主色调 */ --secondary-color: #3498db; /* 次要色调 */ --accent-color: #e74c3c; /* 强调色 */ --success-color: #2ecc71; /* 成功色 */ /* 更多颜色变量... */ }

4. 添加学习资源

在资源库部分添加新的学习资源卡片：

<div class="resource-card"> <h4>资源名称</h4> <div class="resource-meta"> <span class="resource-type">资源类型</span> <span class="resource-difficulty difficulty-beginner">难度</span> </div> <p>资源描述</p> <a href="资源链接" target="_blank" class="resource-link">访问资源</a> </div>
